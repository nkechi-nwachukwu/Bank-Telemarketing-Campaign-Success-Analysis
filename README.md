# 📊 Bank Telemarketing Campaign Success Analysis

## 🎯 Project Overview
This project analyzes a bank telemarketing campaign dataset to identify key drivers of customer conversion, optimize contact strategies, and improve marketing ROI through data-driven insights.

---

## 🎯 Business Problem
Banks often run telemarketing campaigns to promote term deposits, but conversion rates are typically low.  
This project aims to answer:

- What factors influence a customer’s likelihood to subscribe?
- Which customer demographics are most responsive to campaigns?
- How can marketing strategies be optimized to improve conversion rates?

---

## 🛠 Tools Used
- Power BI  
- Power Query (Data Cleaning & Transformation)  
- DAX (Calculated Measures & KPIs)  

---

## 🔄 Approach
- Cleaned and transformed raw dataset using Power Query  
- Handled missing values and standardized categorical variables  
- Built data model in Power BI  
- Created calculated measures using DAX (e.g., conversion rate, campaign success rate)  
- Designed an interactive dashboard with filters for age, job, education, and marital status  
- Analyzed customer segments to identify conversion patterns  

---

## 📊 Key Insights
🔹 Conversion rate improved to 11.3% (vs 3.3%), but still has room for optimization
🔹 Customers aged 26–45 showed the highest response to campaigns
🔹 Admin, technician, and blue-collar jobs recorded the strongest conversions
🔹 First contact was most effective, with diminishing returns from repeated calls
